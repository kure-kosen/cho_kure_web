## Setup
以下を実行すればセットアップは完了するはず。
mysqlに接続できなければ `.env` を編集して自分の環境に合わせてください。
```
git clone git@github.com:kure-kosen/cho_kure_web.git
cd cho_kure_web
bin/setup
```
