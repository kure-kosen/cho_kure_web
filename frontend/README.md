## How to Dev

```bash
$ yarn install

# Commands
$ yarn start  # Start webpack-dev-server, open http://localhost:3000/app
$ yarn dev    # same as $ yarn start
$ yarn build  # Build to /public/assets/build/
$ yarn lint   # Check type and syntax, auto fix
```
